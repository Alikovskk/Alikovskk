ctrl + alt + t - горячая клавиша для открытия терминала
Директория = папка

ls - Показывает все файлы и папки внутри текущей директории
ls test - показывает все файлы и папки внутри папки test
ls -a - показывает также все скрытые файлы и папки
ls -l - показывает более подробную информацию о файлах и папках

cd - переходит в домашнюю папку
cd test/ - переходит в папку test, если такая папка существует в текущей директории
cd .. - переходит на 1 директорию назад
cd ../.. - переходит на 2 папки назад

mkdir test - создает папку test в текущей директории
mkdir test1/test2 - создает папку test2 внутри test1
mkdir test1 test2 test3 - создает папки test1 test2 и test3 в текущей директории

touch test.txt - создает файл test.txt в текущей директории

rm test.txt - удаляет файл test.txt (если такого файла нет в текущей директории, выкинет ошибку)
rm -rf test/ (либо test.txt) - может удалить и папку со всем содержимым, и файл

mv test.txt test/ - перемещает файл test.txt в папку test
mv test.txt lesson.py - переименовывает файл test.txt в lesson.py
mv test1/ test_lesson/ - переименовывает директорию test1 в test_lesson

history - показывает все команды, которые были выполнены в этом терминале

pwd - показывает абсолютный (начиная с корневой папки, то есть папка '/') путь до текущей директории
(print working directory)

cat test.txt - показывает все что находится внутри файла
nano test.txt - открывает файл test.txt в редакторе nano
code . (либо, вместо точки можно написать название директории или файла, и отрыть именно его) - открывает файл или папку в редакторе кода VSCode
